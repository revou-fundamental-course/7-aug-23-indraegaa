Hello guys, Selamat datang di website Kample (Kalkulator Simple) :)

Kample (Kalkulator Simple) adalah website untuk pengguna yang sangat simple untuk menghitung suatu nilai luas dan keliling bangun datar pada segitiga

Segitiga

Luas Segitiga

Cara menghitung Luas Segitiga:

L = 1/2 x a x t

Keterangan:
L = Luas
a = Panjang Alas
t = Tinggi


Hitung Luas Segitiga







Keliling Segitiga

Cara menghitung Keliling Segitiga:

K = S1 + S2 + S3

Di mana :
K = Keliling
S1 = Sudut Segitiga A
S2 = Sudut Segitiga B
S3 = Sudut Segitiga C
